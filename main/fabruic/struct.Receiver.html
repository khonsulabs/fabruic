pub async fn finish(&self) -> Result<()>[src]

Wait for the Receiver part of the stream to finish gracefully.

This can only be achieved through the peer’s Sender::finish or an error.

Errors

  • Error::Read if the Receiver failed to read from the stream
  • Error::AlreadyClosed if it has already been closed

Close the Receiver immediately. To close a Receiver gracefully use finish.

Errors

  • Error::AlreadyClosed if it has already been closed
